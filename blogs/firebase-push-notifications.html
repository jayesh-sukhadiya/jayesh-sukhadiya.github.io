Implementing Firebase Push Notifications in Laravel

Feb 1, 2024 Jayesh Sukhadiya

Introduction

Push notifications are essential for engaging users and providing real-time updates in modern web applications. In this comprehensive guide, we'll walk through implementing Firebase Cloud Messaging (FCM) in a Laravel application.

Prerequisites

  • Laravel 8.x or higher
  • Firebase project setup
  • Composer installed

Step 1: Firebase Project Setup

First, create a new Firebase project and configure the web credentials:

// Initialize Firebase
const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-domain.firebaseapp.com",
  projectId: "your-project-id",
  messagingSenderId: "your-sender-id"
};

Step 2: Install Required Packages

Install the Firebase PHP Admin SDK using Composer:

composer require kreait/firebase-php

Step 3: Configure Laravel

Set up your Firebase credentials in Laravel's configuration:

// config/services.php
return [
    'firebase' => [
        'credentials' => storage_path('firebase-credentials.json'),
    ]
];

Step 4: Create Notification Service

Implement a service class to handle Firebase notifications:

namespace App\Services;

use Kreait\Firebase\Messaging\CloudMessage;

class FirebaseNotificationService
{
    public function sendNotification($token, $title, $body)
    {
        $message = CloudMessage::withTarget('token', $token)
            ->withNotification([
                'title' => $title,
                'body' => $body
            ]);
            
        return $this->messaging->send($message);
    }
}

Step 5: Implement Frontend Integration

Add the necessary JavaScript code to handle FCM on the client side:

// Request notification permission
Notification.requestPermission().then((permission) => {
    if (permission === 'granted') {
        // Get FCM token
        messaging.getToken().then((token) => {
            // Send token to your server
        });
    }
});

Best Practices and Tips

  • Always handle token refresh events
  • Implement proper error handling
  • Store tokens securely in your database
  • Test notifications thoroughly before deployment

Conclusion

With Firebase Cloud Messaging integrated into your Laravel application, you can now send targeted push notifications to your users. This implementation provides a solid foundation for building real-time notification features.
