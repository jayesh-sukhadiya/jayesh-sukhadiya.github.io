Laravel Horizon: Redis Queue Monitoring Made Easy

Feb 16, 2024 Jayesh Sukhadiya

Introduction

Laravel Horizon provides a beautiful dashboard and code-driven configuration for your Redis queues. It allows you to easily monitor key metrics of your queue system such as job throughput, runtime, and job failures.

Prerequisites

  • Laravel 8.x or higher
  • Redis server installed
  • PHP Redis extension

Step 1: Installation

Install Laravel Horizon via Composer:

composer require laravel/horizon

Step 2: Configuration

Publish the Horizon assets and configuration:

php artisan horizon:install
php artisan horizon:assets

Step 3: Environment Setup

Configure your environment variables:

QUEUE_CONNECTION=redis
REDIS_CLIENT=predis
REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

Step 4: Queue Configuration

Configure your queues in the Horizon configuration file:

// config/horizon.php
'environments' => [
    'production' => [
        'supervisor-1' => [
            'connection' => 'redis',
            'queue' => ['default', 'emails'],
            'balance' => 'simple',
            'processes' => 10,
            'tries' => 3,
        ],
    ],
    'local' => [
        'supervisor-1' => [
            'connection' => 'redis',
            'queue' => ['default'],
            'balance' => 'simple',
            'processes' => 3,
            'tries' => 3,
        ],
    ],
],

Step 5: Starting Horizon

Start the Horizon process:

php artisan horizon

Best Practices and Tips

  • Use process supervisors in production
  • Monitor memory usage
  • Configure appropriate timeouts
  • Implement proper error handling

Advanced Features

// Custom Metrics
Horizon::metrics([
    'jobs' => 60,
    'failed_jobs' => 60,
]);

// Job Batching
Bus::batch([
    new ProcessPodcast(Podcast::find(1)),
    new ProcessPodcast(Podcast::find(2)),
])->then(function (Batch $batch) {
    // All jobs completed successfully...
})->catch(function (Batch $batch, Throwable $e) {
    // First batch job failure...
})->finally(function (Batch $batch) {
    // The batch has finished executing...
})->dispatch();

Conclusion

Laravel Horizon provides a powerful and elegant solution for monitoring and managing your Redis queues. By following this guide, you can set up a robust queue monitoring system that helps you maintain and optimize your application's background job processing.
